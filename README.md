# aibind

AI SDK bindings for your favorite UI framework. Type-safe, reactive, streaming-first.

## Packages

| Package | Description |
|---------|-------------|
| [@aibind/svelte](packages/svelte) | AI SDK bindings for SvelteKit — reactive streaming, structured output, agents, and server helpers |

## Development

```bash
pnpm install
pnpm test
pnpm dev        # start demo playground
```
